Element Component

  Element មួយគឺជាបន្សំនៃ open tag មួយ អាចជាមួយនឹង attributes មួយចំនួន តំលៃមួយចំនួននិងclosing tag មួយ។ Element អាចផ្ទុកនូវ elements ។
Tags
  Tag ជាការសំគាល់ចំនុចចាប់ផ្ដើម និងចំនុចបញ្ចប់នៃ element មួយបញ្ជាក់ដោយ សញ្ញា (<>)។ tag មានពីរគឺ Open tag(<tag>) Closing Tag(</tag>)
Example:
  <html></html>
  <h1></h1>
  មាន <tag> មួយចំនួនមិនមាន closing tagទេ ដូច្នេះ tag ប្រភេទនេះគេហៅថា self-closing element រឺ replaced element ដូចជា <img>, <br>, <hr>,<meta>។
Attribute
  Attribute ជាការកំនត់លក្ខណៈរបស់ element មួយ។ Attribute ចែកចេញជាពីរប្រភេទគឺ៖ Attribute NameនិងAttribute Value។
  tags ភាគច្រើនមាន attribute បីប្រភេទដូចខាងក្រោម៖,
Core Attribute
  ➤ id: ឈ្មោះសំគាល់ element ដែលក្នុង document មួយមិនអាចមាន id ដូចគ្នាបាទ
  ➤ class ជាឈ្មោះសំគាល់ class របស់ element ដែលប្រើក្នុង style sheets
  ➤ style: ប្រើសំរាប់កំនត់ style អោយ element
  ➤ title: សំរាប់បង្កើត tooltip ពេល ដាក់ mouse ពីលើ
  Example:
   <h1 title="Heading Text">Hello World!</h1>
   <h1 style="color:red">Hello World!</h1>
Language Attribute
  ✡ dir: A direction sur text insattan ltr (left to right), rtl (right to left)
  ✡ lang ជាភាសាដែលនឹងត្រូវប្រើសំរាប់ attribute និង content របស់ element មួយ។ browser អាចប្រើពត៌មាននេះដើម្បីជំនួយក្នុងការ បកប្រែ content ដើម្បីគោរពនឹងភាព លំអិតដូចជា ប្រព្រឹត្តិកម្មនៃចំណង(ពេលដែល support ដោយ font ណាមួយ រឹតម្រូវការ ដោយភាសាដែលបានសរសេរ), quotation marks, and hyphenation។ ម្យ៉ាងវិញទៀត កម្មវិធី និង search engines ដទៃទៀតអាចប្រើពត៌មាននេះដើម្បីជំនួយក្នុងការ ជ្រើសរើសspell-checking dictionary និងការបង្កើត index។
 Example:
  <h1 dir="rtl" lang="en"> Hello World!</h1>
